# HealthSphere AI

An intelligent health assistant application that analyzes symptoms, predicts diseases, determines urgency levels, and recommends appropriate doctors using Machine Learning and AI.

## üöÄ Quick Start

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd HealthspherAI
   ```

2. **Follow the setup guide**
   - See [SETUP_GUIDE.md](./SETUP_GUIDE.md) for detailed setup instructions

3. **Start the application**
   ```bash
   # Terminal 1: Backend
   cd backend
   npm install
   npm run dev

   # Terminal 2: Frontend
   cd frontend
   npm install
   npm start
   ```
