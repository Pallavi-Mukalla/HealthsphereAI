# HealthSphere AI

An intelligent health assistant application that analyzes symptoms, predicts diseases, determines urgency levels, and recommends appropriate doctors using Machine Learning and AI.

## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd HealthspherAI
   ```

2. **Follow the setup guide**
   - See [SETUP_GUIDE.md](./SETUP_GUIDE.md) for detailed setup instructions

3. **Start the application**
   ```bash
   # Terminal 1: Backend
   cd backend
   npm install
   npm run dev

   # Terminal 2: Frontend
   cd frontend
   npm install
   npm start
   ```

## 📚 Documentation

- **[SETUP_GUIDE.md](./SETUP_GUIDE.md)** - Complete setup instructions for collaborators

## 🛠️ Technology Stack

- **Frontend**: React 18
- **Backend**: Node.js, Express.js
- **Database**: MongoDB Atlas
- **ML Service**: Python, scikit-learn
- **AI**: Google Gemini API

## ✨ Features

- Multimodal symptom input (text, voice, image)
- ML-powered disease prediction
- AI fallback and conflict resolution
- Smart urgency detection
- Proximity-based doctor recommendations
- Dark theme chatbot interface
